## Getting Started

Install the Maizzle CLI:

```
npm install -g @maizzle/cli
```

Scaffold a new project with this starter:

```
maizzle new amp4email
```

Develop locally:

```
maizzle serve
```

Build for production:

```
maizzle build production
```
